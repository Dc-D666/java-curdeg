# 我的通知页面重新设计 - 设计规格

## 设计目标
- 去除所有 emoji，使用纯文字/几何图标
- 卡片式布局紧凑对齐，视觉信息层次分明
- 配色以浅蓝/白/灰色为主，保留原有功能不变

```

│  我的通知          [未读 3]              │  ← 顶栏
├─────────────────────────────────────────┤

├─────────────────────────────────────────┤
│                                         │
│  ┌───────────────────────────────────┐  │
│  │ 今天                              │  │
│  ├───────────────────────────────────┤  │
│  │ [系] 系统通知         刚刚        │  │  ← 通知卡片
│  │ 您的账号已完成安全验证           │  │
│  │ [标记已读]                         │  │

## 2. 顶栏 (Header/Toolbar)

| 元素 | 说明 |
|------|------|
| 标题 "我的通知" | 20px 粗体，`#1f2937` |
| 未读标记 "未读 N" | 右侧圆角药丸标签，蓝色背景 `#2563eb` 白色文字 |

## 3. Tab栏

| 按钮 | 默认状态 | 激活状态 |
|------|----------|----------|
| 全部 N | 灰色 `#f1f5f9` | 蓝色 `#2563eb` 白字 |
| 未读 N | 同上 | 同上 |
| 已读 N | 同上 | 同上 |
| 全部已读 | 右侧操作按钮，`#f8fafc` 背景，hover蓝色 |

## 4. 日期分隔

| 元素 | 样式 |
|------|------|
| 文本 "今天/昨天/星期一/5月21日" | 12px 灰色 `#94a3b8` 加粗，左侧4px蓝色竖条 `#2563eb` |

## 5. 通知卡片

每张卡片为白色圆角 8px 矩形，带 2px 间距，整体结构：

```
┌────────────────────────────────────────────────┐
│ [图标框]  类型文字                 相对时间    │
│           内容摘要（单行，超出省略）           │
│           操作按钮行（hover时显示）             │
└────────────────────────────────────────────────┘
```

### 5.1 图标框

| 通知类型 | 图标文字 | 背景色 | 文字颜色 |
|----------|----------|--------|----------|
| 系统通知 | 系 | `#eff6ff` | `#2563eb` |
| 举报处理通知 | 举 | `#fff7ed` | `#d97706` |
| 帖子审核通知 | 审 | `#f0fdf4` | `#16a34a` |
| 评论回复通知 | 评 | `#ecfeff` | `#0891b2` |
| 新增粉丝通知 | 粉 | `#faf5ff` | `#9333ea` |
| 关注用户发帖 | 关 | `#eef2ff` | `#4f46e5` |

图标框尺寸：40x40px，圆角 8px，居中显示文字。

### 5.2 审核标签 (仅审核通知)

| 状态 | 标签文字 | 背景色 | 文字颜色 |
|------|----------|--------|----------|
| 审核通过 | [通过] | `#f0fdf4` | `#16a34a` |
| 审核未通过 | [未通过] | `#fef2f2` | `#dc2626` |

### 5.3 未读指示器

左侧红色小圆点 `#2563eb`，直径 8px，仅未读通知显示。

### 5.4 操作按钮

| 按钮 | 样式 |
|------|------|
| 标记已读 | 透明背景，蓝色文字 `#2563eb`，1px蓝色边框 |
| 查看原帖 | 透明背景，绿色文字 `#16a34a`，1px绿色边框 |

按钮默认隐藏，卡片 hover 时显示。未读通知默认显示。

## 6. 空状态

│          [ ! ]                │
│          暂无通知              │
│    当您收到通知时会显示在这里  │
│                               │

│                               │
└───────────────────────────────┘
```

## 7. 底栏

居中显示文字 "共 N 条通知 · M 条未读"，12px 灰色 `#94a3b8`。

## 8. 配色方案

| 用途 | 色值 |
|------|------|
| 页面背景 | `#eef2f7` |
| 卡片背景 (未读) | `#ffffff` |
| 卡片背景 (已读) | `#fafbfc` |
| 卡片边框 (已读) | `#f0f0f0` |
| 主色 | `#2563eb` |
| 正文文字 | `#1f2937` |
| 次要文字 | `#4b5563` / `#94a3b8` |
| 分割线 | `#e5e7eb` |

## 9. 交互行为

- 单击卡片：如果未读则标记为已读
- 双击卡片：标记已读 + 打开关联帖子详情
- 右键卡片：弹出上下文菜单（标记已读 / 查看原帖）
- hover卡片：显示操作按钮行，轻微阴影提升

## 10. 功能保持不变

- 三类筛选 Tab（全部/未读/已读）
- 全部已读按钮
- 刷新按钮
- 加载状态（进度指示器）
- 空状态（无通知时显示）
- 右键菜单
- 标记已读单个/批量
- 双击打开关联帖子
- 底部计数文本
