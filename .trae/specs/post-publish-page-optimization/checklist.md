# 发布帖子页面优化 - 验证清单

## 布局验证
- [ ] 页面被清晰地划分为多个区域（基本信息区、内容编辑区、AI助手区、媒体区等）
- [ ] 各区域有明确的视觉分隔（标题、分隔线或其他视觉标识）
- [ ] 控件对齐整齐，间距合理美观
- [ ] 页面与项目现有设计风格保持一致
- [ ] 在不同窗口大小下布局都能正常显示

## AI写作助手集成验证
- [ ] AI写作助手区域使用 TitledPane 或类似可折叠容器
- [ ] AI区域默认状态合理（展开或折叠，取决于用户场景）
- [ ] AI区域可以正常折叠和展开
- [ ] AI功能与主要发布流程不冲突
- [ ] AI生成结果有对比查看功能
- [ ] AI生成的内容可以接受或拒绝

## 图片上传功能验证
- [ ] 支持多张图片的预览展示
- [ ] 图片使用网格布局美观地展示
- [ ] 每个图片有单独的删除按钮
- [ ] 图片可以正常删除
- [ ] 图片上传过程有进度反馈
- [ ] 图片URL管理方式更直观，不再是简单的逗号分隔文本框

## 字数统计功能验证
- [ ] 标题下方显示实时字数统计
- [ ] 内容下方显示实时字数统计
- [ ] 字数统计格式为 `当前字数: X/最大限制`
- [ ] 接近限制时统计标签变为警告色（如红色）
- [ ] 字数统计输入时实时更新

## 草稿保存功能验证
- [ ] 有手动保存草稿按钮
- [ ] 自动定期保存功能正常工作
- [ ] 再次打开页面时会提示恢复草稿
- [ ] 草稿恢复功能正常
- [ ] 发布后草稿被正确清除
- [ ] 草稿保存不影响正常发布流程

## 内容预览功能验证
- [ ] 有预览按钮
- [ ] 点击预览能显示帖子的实际效果
- [ ] 预览包含标题、板块、内容和图片
- [ ] 预览界面美观且与实际帖子展示一致

## 交互反馈验证
- [ ] 表单验证有实时反馈（不只是在发布时）
- [ ] 错误提示友好且清晰
- [ ] 发布过程有进度指示
- [ ] 发布成功后有预览帖子的选项
- [ ] 所有按钮状态在操作过程中正确更新

## 功能完整性验证
- [ ] 原有的发布功能正常工作
- [ ] 原有的取消功能正常工作
- [ ] 原有的图片上传功能正常工作
- [ ] 新添加的所有功能都能正常使用
- [ ] 没有引入新的 Bug

## 用户体验验证
- [ ] 整体操作流程更加流畅
- [ ] 页面布局更加清晰易懂
- [ ] 新增功能直观易用
- [ ] 没有破坏原有的用户习惯
- [ ] 用户反馈应该是正面的
