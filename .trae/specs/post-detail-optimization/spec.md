
# 帖子详情页面优化 - Product Requirement Document

## Overview
- **Summary**: 对校园论坛系统的帖子详情页面进行全面优化，包括布局重构、功能完善和用户体验提升
- **Purpose**: 解决当前页面布局混乱、功能缺失、信息展示不完整等问题，提升用户使用体验
- **Target Users**: 所有论坛用户，包括学生、管理员

## Goals
- 优化页面布局，提升视觉美观度和操作便利性
- 完善功能（评论点赞、图片预览、分享、刷新等）
- 增强信息展示（板块信息、浏览量、统计数据等）
- 提升代码可维护性（组件化重构）

## Non-Goals (Out of Scope)
- 不涉及后端API的重大重构
- 不修改其他页面（如帖子列表、个人中心等）
- 不实现用户主页功能（后续迭代）
- 不实现评论表情功能（后续迭代）
- 不实现夜间模式（后续迭代）

## Background & Context
- 当前使用JavaFX作为前端框架，Spring Boot作为后端
- 数据库已有相关字段（如评论图片、浏览量等），但前端未展示
- 控制器代码超过1100行，维护困难
- 用户反馈操作按钮混乱、信息不完整

## Functional Requirements
- **FR-1**: 重构页面顶部信息栏，增加板块、浏览量、刷新/返回按钮
- **FR-2**: 优化内容区域，AI总结默认收起，内容改用Label展示
- **FR-3**: 重构底部操作区，拆分高频/低频操作按钮
- **FR-4**: 组件化评论列表，实现CommentItem独立组件
- **FR-5**: 实现评论点赞功能
- **FR-6**: 实现评论图片展示功能
- **FR-7**: 实现图片点击放大预览功能
- **FR-8**: 实现分享功能（复制链接）
- **FR-9**: 实现刷新功能
- **FR-10**: 实现评论分页或加载更多
- **FR-11**: 优化违规帖展示和编辑引导

## Non-Functional Requirements
- **NFR-1**: 页面加载响应时间 &lt; 2秒
- **NFR-2**: 代码可维护性提升，控制器行数减少50%以上
- **NFR-3**: 保持与现有API的兼容性
- **NFR-4**: 优化内存使用，避免图片加载导致的内存泄漏

## Constraints
- **Technical**: Java 21 + JavaFX 21 + Spring Boot 3.2.5 + MySQL
- **Business**: 保持现有功能完整性，优先保证核心功能可用
- **Dependencies**: 依赖现有后端API，如需要新增API需与后端协同

## Assumptions
- 后端API基本满足需求，可能需要小的补充（如评论点赞）
- 用户浏览器/桌面环境支持现有JavaFX功能
- 数据库已有评论图片字段（从代码看已有）

## Acceptance Criteria

### AC-1: 顶部信息栏优化
- **Given**: 用户打开帖子详情页面
- **When**: 页面加载完成
- **Then**: 显示完整的顶部信息，包括板块面包屑、返回/刷新按钮、作者信息、统计数据（浏览量、点赞数、评论数、收藏数）
- **Verification**: `human-judgment`

### AC-2: 审核状态标签展示
- **Given**: 帖子有审核状态
- **When**: 页面显示帖子信息
- **Then**: 审核状态以彩色标签形式展示（橙色-审核中，绿色-已通过，红色-违规，黄色-待人工）
- **Verification**: `human-judgment`

### AC-3: 底部操作区按钮分组
- **Given**: 用户查看帖子详情
- **When**: 查看底部操作区
- **Then**: 高频操作（点赞、收藏、分享、举报）直接显示，低频操作（编辑、删除、置顶、加精、AI总结）在"更多"下拉菜单中
- **Verification**: `human-judgment`

### AC-4: 评论使用独立组件
- **Given**: 帖子有评论
- **When**: 查看评论列表
- **Then**: 每条评论使用独立组件渲染，包含头像、作者、内容、图片、操作按钮
- **Verification**: `human-judgment`

### AC-5: 评论点赞功能
- **Given**: 用户登录且不是被禁言用户
- **When**: 点击评论的点赞按钮
- **Then**: 点赞状态切换，点赞数更新
- **Verification**: `programmatic`

### AC-6: 评论图片展示
- **Given**: 评论包含图片
- **When**: 查看该评论
- **Then**: 评论内容下方显示图片缩略图
- **Verification**: `human-judgment`

### AC-7: 图片点击放大预览
- **Given**: 帖子或评论有图片
- **When**: 用户点击图片
- **Then**: 弹出图片预览窗口，支持查看大图
- **Verification**: `human-judgment`

### AC-8: 分享功能可用
- **Given**: 用户登录
- **When**: 点击分享按钮
- **Then**: 弹出分享对话框，可复制帖子链接
- **Verification**: `human-judgment`

### AC-9: 刷新功能正常
- **Given**: 帖子详情页面已打开
- **When**: 点击刷新按钮
- **Then**: 重新加载帖子详情和评论，显示加载状态
- **Verification**: `programmatic`

### AC-10: 违规帖优化展示
- **Given**: 帖子审核状态为"reject"
- **When**: 查看该帖子
- **Then**: 显示醒目的违规标识，展示违规原因和审核建议，提供清晰的编辑重新提交引导
- **Verification**: `human-judgment`
