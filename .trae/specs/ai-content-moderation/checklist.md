
# AI内容审核功能 - Verification Checklist

## 数据库设计验证
- [ ] bbs_post表已添加审核状态字段
- [ ] 审核状态字段包含：pending（审核中）、pass（通过）、reject（违规）、manual（待人工审核）
- [ ] 包含审核结果详情字段（违规类型、置信度、备注等）
- [ ] 包含审核时间字段
- [ ] BbsPost实体类正确映射新字段

## 后端服务验证
- [ ] HTTP客户端依赖已添加
- [ ] AI审核服务能正常调用API
- [ ] 能正确解析API返回的JSON结果
- [ ] API调用失败时有降级处理
- [ ] 异步审核任务不阻塞主流程
- [ ] 审核状态正确更新

## 业务逻辑验证
- [ ] 发帖后初始状态为"审核中"
- [ ] 普通用户只能看到审核通过的帖子

- [ ] 管理员能看到所有帖子
- [ ] 审核通过后帖子全员可见
- [ ] 审核拒绝后帖子仅作者可见
- [ ] 待人工审核帖子仅作者和管理员可见

## 通知系统验证
- [ ] 审核完成后用户收到通知

- [ ] 管理员能查看待审核帖子列表
- [ ] 管理员能通过或拒绝帖子
- [ ] 审核操作有日志记录
- [ ] 审核后用户收到通知

## 前端UI验证
- [ ] 审核状态显示正确

- [ ] 状态标签仅作者和管理员可见
