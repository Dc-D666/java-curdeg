# 用户主页 - 验证检查清单

## 界面设计检查
- [ ] 用户头像正确显示，尺寸合适
- [ ] 用户昵称、个性签名等信息完整显示
- [ ] 统计数据（发帖数、粉丝数、关注数）正确显示
- [ ] 界面布局合理，视觉层次清晰
- [ ] 颜色、字体与现有界面风格一致
- [ ] 关注按钮样式正确，状态切换明显
- [ ] 私信按钮正确显示（查看他人主页时）
- [ ] 发帖列表格式规范，信息完整
- [ ] 分页控件正常显示

## 功能检查
- [ ] 点击帖子详情页作者头像能跳转到用户主页
- [ ] 点击评论区用户头像能跳转到用户主页
- [ ] 用户主页在新标签页正确打开
- [ ] 关注/取消关注按钮点击后状态正确更新
- [ ] 粉丝数在关注操作后同步更新
- [ ] 用户发帖列表正确加载
- [ ] 发帖列表分页功能正常
- [ ] 点击列表中的帖子能跳转到帖子详情
- [ ] 刷新功能正常工作
- [ ] 返回功能正常工作
- [ ] 点击私信按钮显示"功能开发中"提示

## 数据准确性检查
- [ ] 用户基本信息与实际一致
- [ ] 发帖数统计正确
- [ ] 粉丝数统计正确
- [ ] 关注数统计正确
- [ ] 发帖列表只显示审核通过的帖子
- [ ] 帖子信息（标题、内容摘要、时间、点赞数、评论数）正确

## 用户体验检查
- [ ] 页面加载速度合理（< 2秒）
- [ ] 加载过程有适当的提示
- [ ] 错误情况有友好的提示
- [ ] 空状态有适当的提示
- [ ] 按钮点击有反馈
- [ ] 整体操作流畅
